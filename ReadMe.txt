Temis es una aplicación web desarrollada utilizando ASP.NET que proporciona funcionalidades relacionadas con la gestión de abogados y usuarios en un entorno legal. La aplicación está diseñada para facilitar el registro y la gestión de información de abogados, así como para permitir a los usuarios iniciar sesión y acceder a sus perfiles.

Características Principales:

Registro de Abogados: La aplicación permite el registro de nuevos abogados, con información detallada como nombre, apellidos, DUI, especialidad, teléfono, correo electrónico, entre otros.

Inicio de Sesión de Usuarios: Los usuarios pueden iniciar sesión utilizando un nombre de usuario y una contraseña proporcionados durante el registro.

Gestión de Usuarios: Los abogados pueden acceder a una interfaz para ver y actualizar su información personal, como el nombre, la especialidad, el teléfono, etc.

Acceso a Funcionalidades Específicas: Dependiendo del rol del usuario, se proporciona acceso a diferentes funcionalidades. Por ejemplo, los abogados pueden tener acceso a herramientas específicas relacionadas con su práctica legal, mientras que los usuarios pueden tener acceso a consultas legales o solicitar servicios.

Implementación:
El proyecto utiliza ASP.NET Framework con el patrón de diseño MVC (Modelo-Vista-Controlador) para una estructura organizada y mantenible. Se han creado vistas HTML con controles ASP.NET para la presentación de datos y la interacción del usuario. Además, se han definido modelos para representar la estructura de datos del proyecto, como los detalles del abogado y del usuario.

Pasos Realizados:

Se ha diseñado una interfaz de usuario utilizando HTML y controles ASP.NET para capturar y mostrar información relacionada con abogados y usuarios.
Se ha implementado la lógica de negocio para el inicio de sesión de abogados y usuarios, así como la gestión de información personal.
Se ha utilizado una base de datos SQL Server para almacenar la información de los abogados y usuarios.
Se ha integrado la lógica de acceso a datos utilizando ADO.NET para realizar consultas y actualizaciones en la base de datos.
